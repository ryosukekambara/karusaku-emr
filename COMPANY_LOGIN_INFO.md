## 🛠️ トラブルシューティング

### **ログインできない場合**
1. **メールアドレスとパスワードを確認**
   - 大文字小文字を正確に入力
   - スペースが入っていないか確認

2. **サーバーが起動しているか確認**
   ```bash
   npm start
   ```

3. **ブラウザのキャッシュをクリア**
   - Ctrl+F5（Windows）
   - Cmd+Shift+R（Mac）

### **エラーが発生した場合**
1. **コンソールログを確認**
   - F12キーで開発者ツールを開く
   - Consoleタブでエラーメッセージを確認

2. **サーバーログを確認**
   - ターミナルでサーバーのログを確認
   - エラーメッセージを記録
